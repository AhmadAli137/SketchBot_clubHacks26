# Task Assignment Template

Use this template when SketchBot delegates work to a worker agent.

## Worker
- firmware | webapp | backend | test

## Objective
- what the worker should accomplish

## Constraints
- contracts to respect
- files/directories owned
- things not to change

## Deliverable
- expected output
- code, docs, tests, or analysis

## Return format
- completed
- assumptions
- blockers
- suggested next step
