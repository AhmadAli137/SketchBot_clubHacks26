# SketchBot Admin Web

Hosted marketing and administrative site for SketchBot.

## Responsibilities

- Corporate and product-facing website
- Account and team sign-in
- Administrative portal
- Release/update visibility
- Hosted integrations that do not belong in the local runtime

## Development

```powershell
cd apps/admin-web
npm install
npm run dev
```

The site runs on `http://127.0.0.1:3002`.
